# eslint-config-fetp

> FETP Eslint Recommond Configuration.

## Note

please use by [fetp](https://github.com/fetp/fetp)

## Install

```bash
$ npm i @fetp/eslint-config-fetp -g
```

## How to use

`package.json` 

```
{
  "eslintConfig": {
    "extends": "@fetp/eslint-config-fetp"
  },
  "eslintIgnore": [
    "/dev",
    "/output"
  ]
}
```
